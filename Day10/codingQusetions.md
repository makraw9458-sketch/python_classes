### Coding Question 1: Even Numbers with Range
**Write a program that:**
- Takes an integer `n` from the user
- Prints all even numbers from 1 to n
- Also prints the count of even numbers

**Expected Output Example:**
```
Enter a number: 10
Even numbers: 2, 4, 6, 8, 10
Count: 5
```

### Coding Question 3: Student Grade Calculator with Nested Loops
**Write a program that:**
1. Asks how many students there are
2. For each student, asks for their name and 3 subject marks (out of 100)
3. Calculates total, average, and grade for each student
4. Displays a report with all students' information
5. Also finds and displays the student with the highest average

**Grading criteria:**
- A: >= 90
- B: >= 80
- C: >= 70
- D: >= 60
- F: < 60

**Sample Output:**
```
How many students? 2

Student 1:
Name: John
Enter marks for subject 1: 85
Enter marks for subject 2: 90
Enter marks for subject 3: 78

Student 2:
Name: Sarah
Enter marks for subject 1: 95
Enter marks for subject 2: 92
Enter marks for subject 3: 88

--- REPORT ---
John: Total=253, Average=84.33, Grade=B
Sarah: Total=275, Average=91.67, Grade=A

Highest Average: Sarah with 91.67
```
